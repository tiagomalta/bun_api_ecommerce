BODY PARAMETERS

Campo Tipo Descrição
nome String (max 40) alfanumérico e não pode conter caracteres especiais
tipopessoa String (max 1) F: Fisica
J: Juridica
endereco String (max 80)
tipo String (max 1) F: Cons.Final
L: Produtor Rural
R: Revendedor
S: Solidario
X: Exportacao
estado String (max 2)
municipio String (max 60)
cnpj_cpf String (max 14)
nomefan String (max 20) Nome Fantasia
bairro String (max 40)
cep String (max 8)
codmun String (max 5)
endcob String (max 40) Endereço de cobrança
ddd String (max 3)
tel String (max 15) Telefone
endent (opcional) String (max 40) Endereço de entrega
contato String (max 15)
inscr (opcional) String (max 18) Inscricao Estadual
pais String (max 3)
inscrm (opcional) String (max 18) Inscricao Municipal
codpais String (max 5)

BODY PARAMETERS

Campo Tipo Descrição
descricao String (max 70) alfanumérico e não pode conter caracteres especiais
tipo String (max 2)
um String (max 2) Unidade
xorig String (max 1) Origem Produto
locpad String (max 2) Armazem Padrao
grupo String (max 4) Grupo de Estoque
segum String (max 2) Segunda Unidade de Medida
novo String (max 1) Produto Inovacao
conv Decimal (12,4)
tipconv String (max 1) Tipo de Conversao da UM
cc String (max 9) Centro Custo
origem String (max 1) Origem do produto
codant String (max 15) Codigo anterior
picm Decimal (5,2) Alíquota de ICMS
posipi String (max 10) Pos.IPI/NCM
especie String (max 2) Especie Produto da TIPI
exncm String (max 3) Ex-NCM
exnbm String (max 3) Ex-NBM
ipi Decimal(5,2) Aliquota de IPI
aliqiss Decimal(5,2) Aliquota de ISS
picmret Decimal(6,2) % Lucro Calc. Solid.Saida
picment Decimal(6,2) % Lucro Calc. Solid.Entr.
impzfrc String (max 1) Produto Import. Z.Franca
clasfis String (max 2) Classificacao fiscal
grtrib String (max 6) Grupo de Tributacao
irrf String (max 1) Incide imposto renda
cest String (max 7) Cod. Especificador ST
desbse2 String (max 60) Desc SubFamilia
base2 String (max 14) SubSubfamilia
conta String (max 20) Conta Contabil do Produto
contac String (max 20) Conta Contabil de custo
itemcc String (max 9) Item de Conta
xestoq String (max 1) Controle Estoque
xusad1 String (max 1) Usa na NF Entrada
xintpcf String (max 254) Integracao PCF
peso Decimal(11,4) Peso Liquido
pesbru Decimal(11,4) Peso Bruto
cubagem Decimal(12,8) Cubagem
codbar String (max 15) Codigo de Barras
verean String (max 2) Codigo EAN
vigenc Data (max 8) Data Vigencia Inicial - Ex: 01/01/24
codgtin String (max 15) Codigo GTIN
